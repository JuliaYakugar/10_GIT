# Добавить файлы в отслеживание
## git add .

## git add <имя_файла>

# Создаем commit
## git commit -m "<комментарий>"

*Добавить отслеживаемые файлы в commit*

# Слияние веток
## git merge <ветка которую сливаем>

*Сливаем ветку, которую указываем в merge в ту ветку, в которой находимся*
